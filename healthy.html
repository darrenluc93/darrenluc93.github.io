Healthy Restaurants

As a group we were responsible for finding the “Healthiest Chain Restaurants” in Austin. Using the nutritionix API we pulled menu items and ingredients from all chain restaurants in Austin, once items were pulled we compared them to the FDA standards and awarded points if ingredients fell within the FDA standards. We cleaned the data in Python using Pandas. Once the data was cleaned and put in Data Frames we created visuals with Matplotlib and presented to the class.

The github repository can be viewed here: https://github.com/darrenluc93/Healthy-Restaurants
